## 🚀 How to Test

1. **Start Dev Server**:
   ```bash
   npm run dev
   ```

2. **Visit Homepage**:
   ```
   http://localhost:3000
   ```

3. **Test Navigation**:
   - Click header links to jump to sections
   - Test theme toggle (Light/Dark)
   - Test responsive menu on mobile

4. **Test CTAs**:
   - All buttons should direct to `/auth/signup`
   - Click "Login" in header → `/auth/signin`

5. **Test Responsiveness**:
   - Open DevTools (F12)
   - Test on mobile (375px), tablet (768px), desktop (1024px)
